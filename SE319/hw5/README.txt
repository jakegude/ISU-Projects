README

1.  open 'cars.html', the cars table will be displayed
2.  click the 'Year' button, the table will then be sorted by year in acending order
3.  double clicking the 'Year' button, the table will then be sorted by year in descending order
4.  in the make column there is a dropdown menu, by selecting of of the options,
    the only cars displayed will be of the make that is selected
5.  in the option colum there is an increment button in each row, 
    by clicking the button it will increment the number of stock of whichever car is in 
    the row of the button that was click by 1
